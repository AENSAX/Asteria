# 开发规范

## 一、代码规范

### 1. 分层职责

- `src/main` 只处理 Electron 主进程、数据库、文件系统、窗口创建和 IPC。
- `src/preload` 只暴露安全的 `window.asteria` API，不写业务逻辑。
- `src/shared` 只放跨进程共享类型、常量和纯函数。
- `src/renderer/src` 只处理 UI、交互状态和展示逻辑。

### 2. 文件职责

- `views/`：page 内可布局的视图。
- `windows/`：独立窗口页面。
- `components/`：可复用 UI。
- `hooks/`：可复用交互逻辑。
- `utils/`：纯工具函数。

### 3. IPC 与类型

- 新增 IPC 必须先更新 `src/shared/ipc.ts`。
- `preload` 只做转发与类型收口。
- `renderer` 不直接使用 `ipcRenderer`。
- IPC 处理函数只做参数校验、调用 service、返回结果。

### 4. 数据与服务

- 数据库读写统一放在 `database` / repository。
- 跨表流程放在 service，避免堆到窗口逻辑里。
- 导入、导出、迁移、AI、外部系统集成都应在 main service 完成。
- 文件对象和物理文件不是一对一关系，业务默认作用于数据库对象。

### 5. 状态与持久化

- 只影响 UI 的轻量设置可保存在 renderer 本地。
- 业务配置持久化放数据库或 app settings，不要分散到多个地方。
- page、view、布局、搜索和浏览状态由 `App.tsx` 统一保存与恢复。

### 6. 复用优先

- 通用确认、通用进度、通用选择对话框优先用独立窗口。
- `保存`、`应用` 等提交按钮统一使用 `ActionFeedbackButton`。
- 标签输入、右键菜单、分级选择、快捷键录制要抽成共享逻辑。
- 能复用就复用，避免在多个窗口复制同一段交互。

### 7. 后台任务

- 后台任务必须接入主窗口底部工作状态栏。
- 任务状态要能显示当前工作、队列、处理中和完成数。
- 取消必须是“当前文件完成后停下”，不能破坏已完成文件的完整性。
- 任务失败要返回可读错误，不刷无意义的控制台噪声。

### 8. 清理规则

- 删除代码前先用 `rg` 确认没有引用。
- 不做与当前需求无关的大重构。
- 未使用依赖、临时日志、调试输出应及时清理。
- 不要让单个文件无限膨胀，必要时拆 service、组件或工具模块。

## 二、UI 规范

### 1. 总体风格

- 信息密度高、字体小、控件小。
- 偏工具软件，不做展示型、营销型界面。
- 不用渐变、阴影、装饰动画、圆角特效。
- 用纯色、边框、间距表达层级。

### 2. 页面、View、窗口

- page 是工作台标签页。
- view 是 page 内布局模块。
- 独立窗口不进入 page 布局。
- page 标签头可重命名、可中键关闭。
- view 标签头可刷新，刷新不重置上下文。
- 独立窗口标题只在系统标题栏显示，不在内容区重复。

### 3. Dialog

- 通用 dialog 必须是独立窗口。
- 通用 dialog 必须自适应内容大小。
- 通用 dialog 内容区不要再套一层大外框。
- 通用确认、通用进度、删除确认、重复确认都走统一 dialog。

### 4. 列表与布局

- 列表、表格、竖列区在数据少时不要拉伸条目。
- 固定条目高度，空白区域留空。
- 左右分栏必须支持拖拽调整，并保留宽度。
- 可视化网格尺寸和间距必须稳定，不能随数据量抖动。

### 5. 选中与待定

- 选中状态用边框或纯色底表达。
- 待定状态必须和选中、悬浮区分开。
- 支持多选、片选、框选的区域，也必须支持 `Ctrl+A`。
- 框选时要显示明显的矩形框。

### 6. 按钮与表单

- 所有按钮悬浮和按下都要有克制反馈。
- `保存`、`应用` 成功后统一短暂显示绿色反馈。
- 输入框必须有 placeholder。
- 不做不必要的华丽按钮样式。

### 7. 标签、分级、收藏

- 标签颜色在任何地方都要显现。
- namespace 标签颜色由系统稳定分配。
- 标签风格分隔要比普通标签更醒目。
- 分级激活后在文件网格和文件详情左上角显示。
- 收藏是文件状态，收藏按钮放在文件右上角。

### 8. 浏览器与详情

- 浏览 view 必须分页，不允许一页渲染全部文件。
- 浏览 view 缩略图懒加载，快速滚动时要优先当前视野。
- 文件详情使用原始媒体，不读取缩略图缓存。
- 图片、视频、动图都要等比显示，不裁剪、不撑开。
- 视频可缩放、可拖动，点击主体不应误触暂停。

### 9. 快捷键

- 快捷键统一走动作系统，不在业务组件里零散监听。
- `Esc` 关闭独立窗口，主窗口除外。
- `Ctrl+A` 全选当前可选区域。
- 快捷键录制用“按下一次键”方式，不手输字符串。
